# ThePhilanthropist

Bannerlord mod with goal of roleplaying as a philanthropist.

Features:
- Donate to towns/villages to increase prosperity.
- Help rebuild a raided village, higher troop strength can make rebuilding faster.
- Settings menu to modify variables such as prosperity donation cap, gold to prosperity ratio, and change the logic to increase prosperity over time instead of immediately while in game.
